Guide des Commandes pour Control Pad

Ce guide vous aide à configurer les boutons de votre Control Pad pour contrôler votre PC.

Important : Pour que les commandes fonctionnent, l'outil nircmd.exe doit être placé dans le PATH de Windows sur le PC que vous contrôlez. Si vous n'êtes pas sûr, demandez à l'administrateur du PC.

Types d'actions :

1. Contrôle du Volume (NirCmd)

Ces commandes utilisent nircmd.exe pour ajuster le volume global du PC.

  • Augmenter le Volume : nircmd.exe changesysvolume 2000 (Augmente le volume de 2%. Adaptez le nombre pour plus ou moins.)
  • Diminuer le Volume : nircmd.exe changesysvolume -2000 (Diminue le volume de 2%.)
  • Couper/Rétablir le Son (Toggle Mute) : nircmd.exe mutesysvolume 2 (Bascule l'état du son : le coupe s'il est activé, le rétablit s'il est coupé.)

2. Contrôle de la Lecture Média (NirCmd)

Ces commandes simulent les touches multimédia du clavier pour contrôler les lecteurs actifs.

  • Lire / Pause : nircmd.exe sendkeypress media_play_pause
  • Piste Suivante : nircmd.exe sendkeypress media_next_track
  • Piste Précédente : nircmd.exe sendkeypress media_prev_track
  • Arrêter la Lecture : nircmd.exe sendkeypress media_stop

3. Lancer des Applications / Fichiers (Commande Windows `start`)

La commande start de Windows permet de lancer des programmes ou d'ouvrir des fichiers.

  • Lancer Bloc-notes : notepad.exe ou start notepad.exe
  • Lancer Calculatrice : calc.exe ou start calc.exe
  • Lancer OBS Studio : start obs64.exe (Si OBS est dans le PATH)
    OU, si besoin, avec le chemin complet (adaptez le chemin !): start "" "C:\Program Files\obs-studio\bin\64bit\obs64.exe" (Les "" après start sont pour le titre de la fenêtre, importants si le chemin est entre guillemets.)
  • Lancer Discord : start discord.exe (Si Discord est dans le PATH ou installé via le Store)
    OU (adaptez le chemin !): start "" "C:\Users\VotreNomUtilisateur\AppData\Local\Discord\app-1.0.9004\Discord.exe"
  • Ouvrir un Dossier : start C:\Users\VotreNomUtilisateur\Documents\MesProjets
  • Ouvrir une Page Web : start chrome.exe https://www.google.com (Lance dans Chrome si installé)
    OU simplement : start https://www.google.com (Ouvre avec le navigateur par défaut)

4. Raccourcis Clavier (Avancé, nécessite RobotJS sur le serveur)

Ces commandes simulent des frappes de touches (ex: F5, Ctrl+C). Le champ "Shortcut" est prévu pour ça.

Attention : Pour que les raccourcis fonctionnent, la bibliothèque robotjs doit être installée et configurée sur le serveur Node.js. Cela a des pré-requis système (Python, Visual Studio Build Tools). Si ce n'est pas fait, ils ne fonctionneront pas. Pour le moment, utilisez le champ "Command" si possible.
  • Simuler F5 : Entrez F5 dans le champ "Shortcut".
  • Simuler Ctrl+C (Copier) : Entrez Ctrl+C dans le champ "Shortcut".
  • Simuler Win+D (Afficher Bureau) : Entrez Win+D dans le champ "Shortcut".

N'hésitez pas à expérimenter avec d'autres commandes Windows ou les commandes de nircmd.exe pour des actions plus spécifiques. Référez-vous à la documentation officielle de NirCmd pour toutes les options.
